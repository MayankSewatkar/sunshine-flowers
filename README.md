# 🌻 Sunshine Flowers

A simple, modern e-commerce storefront for a pastel-themed flower shop. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Features

- **Product grid** — 12 flower products displayed in a responsive 4-column layout
- **Filter pills** — filter by All, Bouquets, Single Stems, Plants, or Seasonal
- **Live cart counter** — updates in the header as items are added
- **Sticky header** — stays visible while scrolling
- **Pastel design system** — soft pinks, yellows, greens, and purples throughout

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Google Fonts (DM Sans + DM Serif Display)
- No build tools or dependencies required

## Getting Started

Just open the file in a browser:

```bash
open index.html
```

Or serve it locally:

```bash
npx serve .
```
